This modules adds the ability to have customfields on any entity using Morph relationships

## Install

Via Composer

``` bash
$ composer require jlourenco/customfields
```

## Usage

``` php
$skeleton = new League\Skeleton();
echo $skeleton->echoPhrase('Hello, League!');
```
